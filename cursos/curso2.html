Curso 2: Desarrollo Web Básico

En este curso aprenderás a crear páginas web utilizando HTML y CSS. Comenzaremos con la estructura básica de una página y luego aplicaremos estilos para hacerla atractiva y funcional.

Este curso es ideal para personas que quieren iniciarse en el mundo del desarrollo web, crear sitios propios o dar sus primeros pasos hacia una carrera como desarrollador front-end.
